# صدقة جارية — لروح المرحوم مصطفى أبو الدهب 🤍

> ﴿ وَلَا تَحْسَبَنَّ الَّذِينَ قُتِلُوا فِي سَبِيلِ اللَّهِ أَمْوَاتًا ۚ بَلْ أَحْيَاءٌ عِندَ رَبِّهِمْ يُرْزَقُونَ ﴾

صفحة ويب بسيطة ومحترمة، الهدف منها أن تكون **صدقة جارية** لروح المرحوم مصطفى أبو الدهب — رحمه الله.
كل من يفتح الصفحة يقرأ دعاءً أو ذكرًا أو سورة، ويُهدي ثوابها له.

## المحتوى

- 🤲 **أدعية للمتوفّى** — دعوة بعد دعوة، مع زر «التالي».
- 📿 **الأذكار والتسبيح** — عدّاد لمسي لكل ذكر (سبحان الله، الاستغفار، الصلاة على النبي…).
- 📖 **سورة الفاتحة** — على روحه.
- 🌙 **سورة يس** — قلب القرآن.
- ✨ **سورة البقرة** — كاملة.

## مميزات

- يعمل على الموبايل والكمبيوتر (تصميم متجاوب، RTL).
- وضع ليلي/نهاري للقراءة المريحة.
- نص القرآن يُجلب من مصدر موثوق ([AlQuran Cloud](https://alquran.cloud) — رواية حفص/الرسم العثماني) ويُخزَّن على الجهاز ليعمل حتى بدون إنترنت بعد أول فتح.
- زر مشاركة لنشر الصدقة.
- بدون أي مكتبات خارجية — HTML/CSS/JS خالص، ليبقى يعمل للأبد.

## التقنية

صفحة ثابتة (static) على GitHub Pages. لا حاجة لأي خطوة بناء (build).

```
index.html   ← الهيكل
styles.css   ← التصميم
app.js       ← المنطق + بيانات الأدعية والأذكار
```
